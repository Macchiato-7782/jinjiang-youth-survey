# 晋江青年婚育观念调查系统 - 云部署指南

## 方案一：Railway（最简单，有免费额度）⭐推荐

### 特点
- 无需配置服务器，自动部署
- 有免费额度（每月500小时运行时间）
- 自动HTTPS
- 适合测试和小规模使用

### 部署步骤

1. **注册账号**
   - 访问 https://railway.app
   - 用GitHub账号登录

2. **创建项目**
   - 点击 "New Project"
   - 选择 "Deploy from GitHub repo"
   - 把你的代码推送到GitHub仓库
   - Railway会自动检测并部署

3. **配置环境变量**（可选）
   - 在 Railway 面板点击 "Variables"
   - 添加 `PORT=3000`

4. **访问应用**
   - 部署完成后会获得一个 `.up.railway.app` 域名
   - 问卷地址：`https://你的项目名.up.railway.app`
   - 后台地址：`https://你的项目名.up.railway.app/admin`

## 方案二：Render（免费，简单）

### 特点
- 免费套餐永久运行
- 自动HTTPS
- 绑定GitHub自动部署

### 部署步骤

1. **注册账号**
   - 访问 https://render.com
   - 用GitHub账号登录

2. **创建Web服务**
   - 点击 "New +" → "Web Service"
   - 连接你的GitHub仓库

3. **配置**
   - Name: `jinjiang-survey`
   - Runtime: `Node`
   - Build Command: `npm install`
   - Start Command: `npm start`

4. **创建**
   - 点击 "Create Web Service"
   - 等待部署完成

## 方案三：阿里云/腾讯云服务器（最稳定，推荐正式使用）

### 推荐配置
- **服务器**: 阿里云轻量应用服务器 或 腾讯云轻量服务器
- **配置**: 2核2G，带宽3Mbps（约100-200元/年）
- **系统**: Ubuntu 22.04 LTS

### 部署步骤

#### 1. 购买服务器
- 阿里云：https://www.aliyun.com
- 腾讯云：https://cloud.tencent.com
- 选择 "轻量应用服务器"，选Ubuntu系统

#### 2. 连接服务器
```bash
# 使用终端连接（Mac/Linux）
ssh root@你的服务器IP

# Windows使用 PuTTY 或 Xshell
```

#### 3. 安装Node.js
```bash
# 更新系统
apt update && apt upgrade -y

# 安装Node.js 18
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
apt install -y nodejs

# 验证安装
node -v  # 应该显示 v18.x.x
npm -v   # 应该显示 9.x.x
```

#### 4. 上传代码
```bash
# 在服务器上创建目录
mkdir -p /opt/survey
cd /opt/survey

# 方式1：用git克隆（推荐）
git clone https://github.com/你的用户名/仓库名.git .

# 方式2：用scp上传（本地终端执行）
scp -r /Users/kai/Desktop/晋江青年婚育观念调查系统/* root@服务器IP:/opt/survey/
```

#### 5. 安装依赖并启动
```bash
cd /opt/survey
npm install

# 测试启动
npm start
```

#### 6. 使用PM2持久运行
```bash
# 安装PM2
npm install -g pm2

# 启动服务
pm2 start server.js --name "survey"

# 保存配置
pm2 save
pm2 startup

# 查看状态
pm2 status
pm2 logs survey
```

#### 7. 配置Nginx反向代理（推荐）
```bash
# 安装Nginx
apt install -y nginx

# 编辑配置文件
nano /etc/nginx/sites-available/survey
```

粘贴以下内容：
```nginx
server {
    listen 80;
    server_name 你的域名或IP;

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

启用配置：
```bash
ln -s /etc/nginx/sites-available/survey /etc/nginx/sites-enabled/
nginx -t
systemctl restart nginx
```

#### 8. 配置HTTPS（SSL证书）
```bash
# 安装Certbot
apt install -y certbot python3-certbot-nginx

# 申请证书
certbot --nginx -d 你的域名

# 按提示操作，选择重定向到HTTPS
```

## 数据备份

### SQLite数据库位置
数据库文件在：`database/survey.db`

### 备份脚本
```bash
# 创建备份目录
mkdir -p /opt/survey/backup

# 备份命令（添加到crontab定时执行）
cp /opt/survey/database/survey.db /opt/survey/backup/survey_$(date +%Y%m%d).db
```

### 自动备份（每天凌晨2点）
```bash
crontab -e

# 添加行：
0 2 * * * cp /opt/survey/database/survey.db /opt/survey/backup/survey_$(date +\%Y\%m\%d).db
```

## 常见问题

### 1. 端口被占用
```bash
# 查看占用3000端口的进程
lsof -ti:3000

# 杀死进程
kill -9 $(lsof -ti:3000)
```

### 2. 防火墙设置
```bash
# 开放80和3000端口
ufw allow 80
ufw allow 443
ufw allow 3000
ufw enable
```

### 3. 数据库权限问题
```bash
# 确保数据库目录有写入权限
chmod 755 /opt/survey/database
chown -R www-data:www-data /opt/survey/database
```
