# 腾讯云轻量服务器 - 一键部署指南

## 步骤概览
1. 购买腾讯云服务器（新用户免费试用1个月）
2. 连接服务器，执行一键部署脚本
3. 获得访问地址，开始收集问卷

---

## 第1步：购买服务器

### 1.1 打开腾讯云
- 访问：https://cloud.tencent.com/product/lighthouse
- 用微信扫码登录

### 1.2 选择配置
| 项目 | 选择 |
|------|------|
| 地域 | 上海 / 广州（离你近） |
| 镜像 | Ubuntu 22.04 LTS |
| 套餐 | 2核2G 4M带宽 |
| 时长 | 1个月（新用户免费）或1年（约100元） |

### 1.3 付款
- 微信支付即可
- 新用户有免费试用额度

## 第2步：连接服务器

购买后，在控制台找到服务器：
- **方式1**：点击"登录"（网页版终端）
- **方式2**：本地终端执行：
  ```bash
  ssh ubuntu@你的服务器IP
  ```
  密码在控制台查看

## 第3步：一键部署（复制粘贴执行）

连接服务器后，把下面这整段复制粘贴进去：

```bash
#!/bin/bash
set -e

echo "========== 开始部署晋江青年婚育观念调查系统 =========="

# 更新系统
echo "[1/8] 更新系统..."
sudo apt update && sudo apt upgrade -y

# 安装Node.js 18
echo "[2/8] 安装Node.js..."
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install -y nodejs
node -v && npm -v

# 安装Git和PM2
echo "[3/8] 安装Git和PM2..."
sudo apt install -y git
sudo npm install -g pm2

# 克隆项目
echo "[4/8] 下载项目代码..."
cd /opt
sudo rm -rf jinjiang-youth-survey 2>/dev/null || true
sudo git clone https://github.com/Macchiato-7782/jinjiang-youth-survey.git
cd jinjiang-youth-survey

# 安装依赖
echo "[5/8] 安装依赖..."
sudo npm install

# 创建数据库目录
echo "[6/8] 配置数据库..."
sudo mkdir -p database
sudo chmod 777 database

# 启动服务
echo "[7/8] 启动服务..."
sudo pm2 delete survey 2>/dev/null || true
sudo pm2 start server.js --name "survey"
sudo pm2 save
sudo pm2 startup

# 安装配置Nginx
echo "[8/8] 配置Nginx..."
sudo apt install -y nginx

sudo tee /etc/nginx/sites-available/survey > /dev/null <<'EOF'
server {
    listen 80;
    server_name _;
    
    client_max_body_size 10M;
    
    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_cache_bypass $http_upgrade;
    }
}
EOF

sudo rm -f /etc/nginx/sites-enabled/default
sudo ln -sf /etc/nginx/sites-available/survey /etc/nginx/sites-enabled/survey
sudo nginx -t && sudo systemctl restart nginx

# 开放防火墙
sudo ufw allow 80/tcp
sudo ufw allow 443/tcp
sudo ufw allow 22/tcp
sudo ufw --force enable 2>/dev/null || true

# 获取IP
IP=$(curl -s ip.sb)

echo ""
echo "=========================================="
echo "✅ 部署完成！"
echo "=========================================="
echo "📝 问卷地址：http://$IP"
echo "📊 后台地址：http://$IP/admin"
echo "=========================================="
echo ""
echo "常用命令："
echo "  查看状态：sudo pm2 status"
echo "  查看日志：sudo pm2 logs survey"
echo "  重启服务：sudo pm2 restart survey"
echo "=========================================="
```

## 数据备份（重要！）

```bash
# 手动备份数据库
sudo cp /opt/jinjiang-youth-survey/database/survey.db ~/backup_$(date +%Y%m%d).db

# 设置自动备份（每天凌晨2点）
echo '0 2 * * * root cp /opt/jinjiang-youth-survey/database/survey.db /root/backup_$(date +\%Y\%m\%d).db' | sudo tee -a /etc/crontab
```

## 绑定域名（可选）

### 1. 购买域名
- 腾讯云搜"域名注册"
- 买个.cn域名（约30元/年）

### 2. 解析到服务器
- 在腾讯云DNS解析，添加A记录指向服务器IP

### 3. 配置HTTPS
```bash
# 安装Certbot
sudo apt install -y certbot python3-certbot-nginx

# 申请证书（替换yourdomain.com为你的域名）
sudo certbot --nginx -d yourdomain.com

# 按提示选择重定向到HTTPS
```
