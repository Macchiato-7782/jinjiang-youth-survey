# 新时代县域青年婚育观念现状及影响因素调查系统

**主题**: 以晋江青年群体为例的婚育观念调查研究

## 系统简介

本系统是一个完整的调查问卷平台，采用苹果风格设计，包含：
- **前端问卷页面**: 响应式设计，支持移动端和PC端，纯白主题
- **后端API服务**: 基于Node.js + Express + SQLite
- **后台管理系统**: 数据可视化、统计分析、交叉分析、数据导出

## 问卷设计（27道题，预计6-8分钟）

### 第一部分：基本信息（6题）
1. 性别
2. 年龄（18-22/23-26/27-30/31-35岁）
3. 最高学历
4. 职业类型
5. 月均收入
6. 居住情况（自有住房/租房独居/与父母同住/其他）

### 第二部分：恋爱观（4题）
7. 感情状况（单身无经历/单身有经历/恋爱中/已婚/离异丧偶）
8. 认识异性渠道（多选）
9. 择偶最看重因素（多选，最多3项）
10. 对异地恋的态度

### 第三部分：婚姻观（5题）
11. 结婚是否是人生必选项
12. 理想结婚年龄
13. 对婚前同居的态度
14. 晋江婚嫁习俗的影响
15. 维系婚姻最重要的因素（多选）

### 第四部分：生育观（5题）
16. 生育意愿（5级：明确想要到完全不想要）
17. 希望生育子女数
18. 对孩子性别的偏好
19. 生育顾虑（多选）
20. 对三孩政策的态度

### 第五部分：家庭与未来（7题）
21. 婚后是否与父母同住
22. 家务劳动分配方式
23. 事业与家庭的平衡
24. 影响婚育决策的主要因素（多选）
25. 晋江地域特色对婚育观念的影响
26. 政府应优先采取的措施（多选）
27. 其他建议（开放题）

## 功能特性

### 前端设计
- **苹果风格界面**: 纯白主题、毛玻璃导航、精致动画
- **爱情元素融入**: 各部分配有浪漫引言
- **分步式问卷**: 5个部分渐进式填写
- **响应式设计**: 完美适配移动端和桌面端

### 后台管理功能
1. **数据概览**: 总回答数、今日统计、趋势图表
2. **人口特征分析**: 性别、年龄、学历、职业分布
3. **恋爱观分析**: 择偶标准、认识渠道可视化
4. **婚姻观分析**: 结婚态度、婚姻维系因素
5. **生育观分析**: 生育意愿、子女数量偏好
6. **家庭观念分析**: 同住态度、家务分配
7. **影响因素分析**: 核心因素识别、政策建议排序
8. **交叉分析**: 支持自定义维度交叉分析
9. **原始数据管理**: 查看详细回答、删除、导出CSV

## 安装与运行

### 系统要求
- Node.js >= 14.0
- npm >= 6.0

### 安装步骤

1. 进入项目目录
```bash
cd 晋江青年婚育观念调查系统
```

2. 安装依赖
```bash
npm install
```

3. 启动服务
```bash
npm start
```

或使用开发模式（自动重启）
```bash
npm run dev
```

4. 访问系统
- 问卷地址: http://124.222.163.91:3000
- 后台管理: http://localhost:3000/admin（本地）

### 数据导出

后台管理页面支持一键导出所有数据为CSV格式，可直接用Excel打开分析。

## 技术栈

- **前端**: HTML5 + CSS3 + JavaScript (苹果设计风格)

- **数据库**: SQLite3

## 研究意义

本问卷设计基于以下研究目标：

1. **全面了解晋江青年人口特征**
   - 年龄、教育、职业、收入、居住状况

2. **深入探究恋爱观念与行为**
   - 择偶标准、认识渠道、异地恋态度

3. **系统分析婚姻观念现状**
   - 结婚必要性、理想婚龄、婚前同居、婚俗影响

4. **准确把握生育意愿与顾虑**
   - 生育打算、子女数量、性别偏好、政策态度

5. **研究家庭观念与代际关系**
   - 同住态度、家务分工、事业家庭平衡

6. **识别影响因素并提出政策建议**
   - 经济/社会/文化因素分析
   - 为优化县域青年婚育环境提供数据支持

## 数据安全

- 所有数据存储在本地SQLite数据库
- 问卷采用匿名方式，不收集个人身份信息
- 支持数据备份和导出
